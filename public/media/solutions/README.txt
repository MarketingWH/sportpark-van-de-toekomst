Zet hier per oplossing lokale foto's of video's neer.

Praktische werkwijze:
- plaats je bestand in deze map, bijvoorbeeld:
  /public/media/solutions/parkeerplaats-ecoraster-01.jpg
- gebruik daarna in src/data/solutions.js als eerste media-item:
  { type: 'image', src: '/media/solutions/parkeerplaats-ecoraster-01.jpg', alt: '...', label: '...' }

Belangrijk:
- het eerste item in de media-array wordt automatisch gebruikt
  in het kleine hotspot-kaartje op de sportparkfoto.
